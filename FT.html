Fog computing

Fog computing or fog networking, also known as fogging, is an architecture that uses edge devices to carry out a substantial amount of computation, storage, and communication locally and routed over the internet backbone.

Tree-based fog computing - TBFC

Fog nodes are hierarchically structured in a height-balanced tree

An application process is assumed to be a sequenceof subprocesses

Every node at each level is equipped with the same subprocess

Each node receives input data from child nodes and performs the subprocess onthe data

    • Edge Node

It is shown the energy consumption and execution time of nodes are shown to be reduced in the TBFC model compared with the cloud computing model

SMPR

Algorithm

ME (minimum energy) algorithm 
                            
                                Input: fR = faulty node
                                Output:fU = new parent node
                                E=∞;
                                foreach candidate fog nodefS in cn(fR) do
                                    NTES=TES(iS+iR);
                                    if NTES < E then
                                        U = S;
                                        E = NTES;
SMPR (selecting multiple parents for recovery) algorithm 
                            
                                Input: fR = faulty node
                                F=ch(fR);
                                while F != φ do
                                    select a disconnected node fD in F whose sizeo Dof output data is maximum;
                                    C = cn(fR);
                                    while C !=φ do
                                        select a candidate node fU where TEU(iU+oD) is minimum in C;
                                        connect fD to fU;
                                        ch(fU) = ch(fU) ∪ {fD};
                                        pt(fD) = {fU};
                                        iU = iU + oD;
                                        C = C - {fU};
                                    F = F - {fD};
